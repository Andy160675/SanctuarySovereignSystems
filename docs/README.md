# Documentation

This directory contains documentation and governance templates for SanctuarySovereignSystems.

## Contents

- `templates/` - Governance templates for traceability, phase gates, threat modeling, etc.

## Governance Templates

The templates in this directory support:

- Requirements traceability
- Phase gate reviews
- Security threat modeling
- Operational runbooks
- Assumption tracking
- Evidence management

See individual template files for usage instructions.
